Hello! This Python Math Project I have created operates using Python's Tkinter GUI library

The application is able to support a multitude of functions such as:
3D vector addition/multiplication
Matrix Multiplication (up to a 10x10 matrix)
Binomial & Normal distribution modelling
Basic calculus functions

This project is still a work in progress, and I plan to expand it's scale by including more useful math modules in the forseeable future!
